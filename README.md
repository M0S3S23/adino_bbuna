# Professional Portfolio & Admin Dashboard

A modern, high-performance portfolio website built with **Next.js 15** and **Supabase**. This project features a secure, real-time admin dashboard that allows the owner to update their professional identity, work history, and academic background without touching a single line of code.

## 🛠 Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Database & Auth:** [Supabase](https://supabase.com/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)

## Key Features

- **Real-time Admin Panel:** Update profile details, experience, and education with instant sync.
- **Secure Authentication:** Protected routes ensuring only authorized users can modify data.
- **Image Hosting:** Integrated Supabase Storage for profile picture uploads.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **Shared Management:** Configured for collaborative access between developer and client.
